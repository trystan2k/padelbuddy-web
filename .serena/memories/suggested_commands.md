# Suggested Commands — Padel Buddy Web

## Development
```bash
pnpm install        # Install dependencies
pnpm dev            # Start local dev server on port 3000
pnpm build          # Production build
pnpm preview        # Preview production build
```

## QA Gate
```bash
npm run complete-check   # (currently a placeholder echo)
```

## Git
```bash
git status
git log --oneline -10
git diff HEAD
```

## GitHub
```bash
gh issue view <number>
gh pr create --title "..." --body "..."
```

## Node Version (managed by mise)
```bash
mise use node@24.14.0
```
