# Padel Buddy Web

Minimal TanStack Start client-only scaffold using pnpm.

## Requirements

- Node `24.14.0`
- pnpm

Node version source of truth is `package.json` `engines.node` and is compatible with `mise`.

## Local Development

```bash
pnpm install
pnpm dev
```

## Production Build

```bash
pnpm build
```
